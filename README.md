# YAML 可视化工具

一个现代化的 YAML 配置文件可视化编辑器，支持表单式编辑和按 key 字母顺序排序。

## 功能特性

- 📄 **文件上传**: 支持拖拽或点击上传 YAML 文件
- 📝 **表单式编辑**: 以直观的表单形式展示和编辑 YAML 配置
- 🔤 **智能排序**: 一键按 key 字母顺序排序所有配置项
- 💾 **保存功能**: 将编辑后的配置保存为 YAML 文件
- 🎨 **现代化 UI**: 美观的用户界面，支持嵌套结构展开/折叠

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用方法

1. 启动应用后，点击或拖拽上传 YAML 文件
2. 文件加载后，可以：
   - 点击展开/折叠按钮查看嵌套结构
   - 直接编辑值
   - 修改 key 名称
   - 添加新的键值对或数组项
   - 删除不需要的配置项
   - 点击"排序"按钮按字母顺序排序所有 key
   - 点击"保存"按钮下载编辑后的 YAML 文件
